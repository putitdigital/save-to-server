    Getting Started with Flowit

    Use this quick guide to set up your first reliable sync in a few minutes.

    1. Configure Settings

    In the main Flowit window, open Settings and set:
    LOCAL_SOURCE: your local folder path.
    DEST_SUBPATH: destination path under your SMB share.

    2. Run and Verify

    Click Sync Now to trigger an immediate sync. Then use Check Status to confirm mount and sync readiness.

    3. Monitor Logs

    Open the admin dashboard and enter the admin code to view sync logs. You can also use Help > Open Sync Logs Folder from the menu.

    4. Automate

    Use Start Sync.command for scheduled background sync, and Stop Sync.command when you want to pause it.
